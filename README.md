# obi-macOS

A comprehensive macOS setup and customization guide for DevOps/SRE engineers.

Covers everything from creating bootable USB installers to configuring development tools, shell environments, and productivity workflows on macOS.

## Topics

- Bootable USB / ISO creation for macOS Sonoma
- System restore and backup workflows
- Development environment setup
- Shell and terminal customization
- Productivity tool configuration

## Usage

Open `index.html` in a browser to view the full guide, or visit the [GitHub Pages site](https://bowang168.github.io/obi-macOS/).
